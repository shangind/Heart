# **Проект предсказания риска сердечных приступов**

## Описание проекта

Требуется разработать модель машинного обучения. Доступен обучающий датасет и тестовый датасет, не содержащий правильных ответов.

Задачи проекта

- Исследование датасета (предполагается поиск дополнительной информации для лучшего понимания природы данных)
- Предобработка данных. Некоторые колонки потребуется преобразовать к нужному типу данных.
- Обучение модели
- Подготовка предсказания на тестовой выборке

## Структура проекта

	create_model.ipynb -  Jupiter Notebook с исследованием, обучением модели, сохранением модели в pkl и получением predictions.csv на тестовой выборке с выводами
	test.py- Тестовое приложение
	requirements.txt - Зависимости для приложения fast api
	best_model.pkl - сохраненная модель
	 preprocessor.pkl - сохраненный препроцессор
	 CSV_Prediction.py - FastAPI приложение
	datasets/ - данные для обучения и тестирования модели

## Порядок запуска FastAPI приложения:

В командной строке

1. pip install -r requirements.txt
2. uvicorn CSV_Prediction:app --reload --host 0.0.0.0 --port 8000
3. перейти в браузере http://localhost:8000/ выбрать файл для загрузки datasets/heart_test.csv
